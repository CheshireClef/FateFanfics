# 【主神组】烟与酒

> 烟雾镜因陀罗的黑帮pa，吔我主神组邪教啦！

当特斯卡特利波卡拎着一瓶酒迈进因陀罗家的庭院时，迎面撞上的是一管正对着他的枪口。

“什么，原来是你啊。”因陀罗刘海阴影下的眼中闪烁着蓝色的寒光，他扫了特斯卡特利波卡一眼便收回了枪口。他银白色的齐整的短发在月光下镀着一层银辉，在他的脚边，庭院雅致的碎石小径上，倒卧着一个不知名的男人。月光照亮着一切，也平等地映亮了男人身下的血泊。

特斯卡特利波卡不禁吹了声口哨。“看来我来得不是时候？”

“不，你来得正是时候。来帮我搭把手。”

金色长发的男人耸耸肩，将拎着的酒放在房子门口的台阶上，然后默契地和因陀罗一起将杀手的尸体拖到花园里一丛狗牙花的阴影下，远离任何可能的视线。特斯卡特利波卡跟着因陀罗在庭院侧面的灌溉水龙头处简单地洗了下手，然后一同爬上几级台阶，穿过房子的正门。

“这是什么东西？”

大门合页所在的门缝被弯曲的透明塑料片从两侧遮挡着。因陀罗回头看了看特斯卡特利波卡手指向的地方。“用来防止门缝夹孩子手指。”他接过特斯卡特利波卡手中的酒瓶，径直进了厨房，出来时手里已多了两只玻璃杯。因陀罗在特斯卡特利波卡边上的沙发上坐下，给自己和他倒上一杯酒，然后仰头一饮而尽。像潜藏在乌云深处的电光，怒火在因陀罗的举手投足之间悄然燃烧着。

“外头那个倒霉蛋，就自己一个人来对付你吗？”
